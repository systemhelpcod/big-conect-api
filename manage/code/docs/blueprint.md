# **App Name**: BIG Connect

## Core Features:

- Visual Workflow Editor: Drag-and-drop interface for designing workflows with connectable nodes.
- Node Execution: Ability to execute individual nodes and entire workflows for testing and deployment.
- Pre-built Node System: Library of pre-built nodes including Webhook, HTTP Request, Function (JavaScript), Delay, Set, WhatsApp API, and database connectors (MySQL, Mongo, Postgres).
- Webhook System: Generates unique URLs, supports test mode with data capture and preview, and handles GET/POST/PUT/DELETE methods.
- Workflow Execution Monitoring: Detailed logs, visualization of data flow between nodes, and debugging options.
- Scheduled Workflow Execution: Enables users to schedule workflows using cron expressions for automated execution.
- AI Node Suggestion: AI-powered tool suggests the most appropriate node for the current workflow context.

## Style Guidelines:

- Primary color: Tech green (#4FCE5D) to reflect automation and connectivity.
- Background color: Dark gray (#242526) for a modern, dark-themed interface.
- Accent color: Purple/pink (#C737A5) as a secondary accent, drawing inspiration from Instagram/Facebook, used sparingly to highlight key elements and interactions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, for a modern, readable interface.
- Use minimalist, vector-based icons to represent nodes and actions.
- Canvas-style layout for workflow design with zoom and drag capabilities.
- Subtle animations for node connections and data flow visualization.
